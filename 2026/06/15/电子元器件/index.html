1. 前序知识

1.1 原子与电子层

  • 电子层：电子大致的运动范围。
  • 每层电子数量：2 * n * n，表示每一层理论上的极限值。
  • 最外层 8 电子稳定结构。

1.2 材料导电性的来源

  • 材料为什么有些能导电，有些不能导电：
    • 碳原子构成的石墨：每个碳原子最多和三个碳原子组合，每个碳原子有一个独立电子，可以随意移动。
    • 金刚石中每个碳原子和四个碳原子组合，没有自由电子。

4. 电阻器

4.1 固定电阻

4.1.1 线绕电阻

  1. 精度较高，最高可达 +-0.01%。
  2. 能承受较大功率。
  3. TCR 较低：随温度变化的程度较低。
  4. 存在寄生电容和寄生电感，不适合高频场景使用，例如 Wi-Fi、蓝牙、卫星通信。

4.1.2 碳膜电阻

  1. 精度较低，一般为 5%。
  2. 功率范围：1/8 W ~ 5 W。
  3. TCR 一般。
  4. 负温度系数：随温度升高，阻值减小。

4.1.3 金属膜电阻

  1. 性能稳定。
  2. 精度较高，一般为 1%，可以达到 0.01%。
  3. 功率范围：50 mW ~ 20 W。
  4. TCR 较低。

4.1.4 贴片电阻

  • 厚膜电阻。
  • 薄膜电阻。

4.1.5 金属铂电阻

  1. 性能极稳定。
  2. 精度极高：一般为 0.01%，最高为 +-0.001%。
  3. 成本很高。
  4. TCR 极低。
  5. 低噪声、耐静电、低感、低容。

4.2 可调电阻（电位器）

  • 类型：滑动式、旋转式。
  • 组成：引脚、滑片、电阻材料。

4.3 特种电阻

4.3.1 保险丝（熔断电阻）

  1. 额定电流：能够通过保险丝正常工作的最大电流。
  2. 熔化特性：快速保险丝、延迟保险丝。
  3. 分断能力：能够安全断开电流的最大能力。
  4. 耐压值：断开后不会产生电弧的电压值。

4.3.2 光敏电阻

  • 光照越强，阻值越小。
  1. 亮电阻。
  2. 暗电阻。
  3. γ 值：阻值与光线变化之间的系数，γ 越大，对光照越敏感。

4.3.3 压敏电阻

  • 作用：保护后方电路。
  1. 压敏电压：随着电压升高，压敏阻值出现变小趋势的拐点电压。
  2. 最大连续工作电压。
  3. 最大限制电压。

4.4 阻值识别

  1. 直标：RKM，阻值 + 精度。
  2. 三位数字代码：一般用于精度为 ±5% 的电阻。
    • 前两位是有效数字。
    • 第三位表示在有效数字后面所加 0 的个数。
    • 字母 R 表示小数点。
    • 示例：
      • 101 -> 100 Ω
      • 151 -> 150 Ω
      • 3R3 -> 3.3 Ω
  3. 四位数字代码：一般用于精度为 ±1% 的电阻。
    • 前三位是有效数字。
    • 第四位表示在有效数字后面所加 0 的个数。
    • 字母 R 表示小数点。
    • 示例：
      • 1001 -> 1000 Ω
      • 10R0 -> 10 Ω
      • 1202 -> 12 kΩ
  4. 精密标注：一般用于精度为 ±0.1% 的电阻。
    • 由两位数字加一位字母表示。
    • 前两位数字标识的阻值要通过查表获得。
    • 最后一位字母表示倍数关系，也需要查表获得。
    • 示例：50B -> 3240，其中 50 和 B 都要查表。
  5. 色环：四色环、五色环。

4.5 阻值标准

  • 10% 精度：E-12 标准。
  • 20% 精度：E-6。
  • 2% 精度：E-48。
  • 1% 精度：E-96。

5. 电容器

重点：电容器隔直通交。

5.1 电容基本特性

  • 流过电容的电压不能突变。
  • 充电速度与电容大小和充电电流有关。
  • 电容充电常数：
  • 3 ~ 5 个 t 值时间可认为电容电压约等于电源电压，也就是电容充满。

5.2 电容储能特性

  • 上电延迟：通过 t 值时间来控制。
  • 关断延时。
  • 滤波：稳压。

5.3 电容器和电池区别

  • 电容器储能过程是物理变化，电池是化学变化。
  • 电容器充电、放电很快，电池较慢。
  • 电容器存储电量较少。
  • 电容自放电率较高。
  • 电容：快速储能与高效释放。
  • 电池：持续稳定。

5.4 容抗

5.4.1 为什么可以隔直通交

  1. 电容的结构决定了“隔直”。
    • 电容的核心结构非常简单：两块相互绝缘的金属极板，中间是绝缘介质。
    • 因为两极板是物理隔绝的，没有直接的导线通路，所以在稳定不变的直流电 DC 面前，它就是一个断路状态。电子无法直接穿越，电流也就无法持续流过，这就是隔直的根本原因。
  2. 电容的充放电实现了“通交”。
    • “通交”的能力，来自于电容在电压变化时发生的充放电过程。
    • 虽然电子不能穿越绝缘层，但可以被“推”到极板上。
    • 电压增大时，正向充电：电源正极从 A 极板拉走电子，同时向 B 极板推入等量电子。这个电子的定向移动，在外电路就形成了电流。
    • 电压减小时，反向放电：之前堆积的电子会被推回原位，形成反向电流。
    • 交流电持续变化时：交流电 AC 的大小和方向就在不断交替变化，导致电容不断地正向充电、反向放电。虽然电子始终没有穿越电容，但外电路上却一直有来回流动的电流，效果上就像电流“通过了”电容一样。

5.4.2 容抗变化规律

  • 电容固定，频率越大，容抗越小。
  • 频率固定，容值越大，容抗越小。

5.5 阻抗、容抗、感抗

  • 阻抗：电路中总的阻碍，由电阻、容抗、感抗构成。
  • 容抗：由电容产生的对电力的“动态阻碍”，随交流电的频率增加而减小。
  • 感抗：由电感产生的对电流的“动态阻碍”，随交流电的频率增加而增加。

5.6 RC 低通滤波

  • 作用：正常让低频信号通过滤波电路，衰减高频信号。
  • 截止频率：低频、高频的界定值由滤波电路中的 R 和 C 确定。
  • 应用：在电路中可以防止高频噪声的干扰。

原理

  • 分压原理：
    • Vout 和电容并联，电压值相等，二者分流。
    • 当输入频率越大，容抗越小，分压越小，out 的分压越小。
    • 当输入频率越小，容抗越大，分压越大，out 的分压越大。
  • 电容充放电特性：
    • 当输入频率很高，电容充放电速度跟不上交流电变化速度，经过电容后的频率就会衰减，也就是高频衰减。
    • 当输入频率很低，电容充放电速度跟得上，经过电容的信号频率就会和输入频率一致，也就是低频通过。

5.7 RC 高通滤波

原理

  • 分压原理：
    • Vout 和电容串联，电流值相等，二者分压。
    • 当输入频率越大，容抗越小，分压越小，Vout 越大。
    • 当输入频率越小，容抗越大，分压越大，Vout 越小。
  • 电容充放电特性。

作用

  • 消除工频干扰。

5.8 电容滤波

  • 二极管整流：将交流电 -> 直流电。
  • 半波整流：一个二极管。
  • 全波整流：整流桥，四个二极管相连。

空气电容

  • 以空气作为电介质的电容器，通常以可调电容存在。

  • 静电双层电容器。
  • 电化学赝电容器。
  • 混合电容器。

6. 电感器

单位：亨利，H。

6.1 电感基本特性

  • 流过电感的电流不能突变。
  • 电感值：抑制电流变化的能力，记作 L。
  • 电感是电能 -> 磁能的器件。
  • 电感器通过电能与磁能之间的相互转化，实现平滑电流，对电流有“来拒去留”的反应。
  • 如果电感所在电路的电阻突然变得很大，例如突然将开关断开，电感会短暂维持原来电流大小，感生出很大的电压，这些电压会击穿器件。
  • 所以要考虑电感的续流电路。

6.3 通直隔交

  • 电感值固定，频率越大，感抗越大。
  • 频率固定，电感值越大，感抗越大。

6.4 电感低通滤波

原理

  • 分压：out 和电感串联分压。
  • 当输入高频信号，感抗大，分压大，后方 out 分压小，电流小。
  • 当输入低频信号，感抗小，分压小，后方 out 分压大，电流大。

6.5 电感高通滤波

原理

  • 分压：out 和电感并联分流。
  • 当输入高频信号，感抗大，分压大，后方 out 分压大。
  • 当输入低频信号，感抗小，分压小，后方 out 分压小。

6.6 RC 低通滤波 VS LC 低通滤波

  • RC 低通滤波适用于电流小的信号传输电路中。
  • LC 低通滤波用于电流大的电源线中。

6.7 储能

8. 蜂鸣器

  • 蜂鸣器是能够发出声音的电子器件。

8.1 有源蜂鸣器与无源蜂鸣器

有源蜂鸣器

  • 内部振荡源频率固定，音调固定。

9. 半导体

9.1 本征半导体

  • 本征半导体是电中性的。
  • 如硅晶体在高温下，能从绝缘体变成半导体，这是材料的天然物理特性，不需要掺杂其他成分。
  • 本征激发：指在本征半导体中，由于外部的热能、光能、电场等激励，使原本处于价带的电子被激发到导带，从而形成电子 + 空穴的现象。

9.2 N 型半导体

  • N 型半导体是电中性的。
  • 因为带来了自由电子，所以叫 N 型。
  • 向硅晶体中掺入适量磷原子，每个磷原子提供一个自由电子，能让半导体导电性更好，也让其导电性更好控制。
  • 电子是多子，空穴是少子。

9.3 P 型半导体

  • P 型半导体是电中性的。
  • 向硅晶体中掺杂硼原子，形成一个空穴。
  • 空穴是多子，电子是少子。

9.4 PN 结

  • 导通时电流方向为 P -> N。
  • 内建电场。

伏安特性曲线

  1. PN 结正偏时，有导通压降。
  2. PN 结反偏时，有漏电流。
  3. PN 结反向电压过高时会产生击穿。
    • 击穿：
    • 电击穿。
    • 热击穿：产生于电击穿。

10. 二极管

  • 二极管由一个 PN 结构成，具有最大反向耐压值。
  • 二极管两端的电压不能超过二极管的导通压降，硅管约等于 0.7 V，否则剩下的电压会产生大电流击穿二极管，除非接适当的电阻。
  • 漏电流：即使二极管反接，也会有很小一部分电流通过，这就是漏电流。
  • 二极管的钳位功能：保护电路。
  • 钳位原理：利用二极管导通后两端电压基本恒定不变的特性，实现对信号线进行钳位。
  • 具体做法：将二极管和电路并联，实现那一段电路的电压被二极管钳位在恒定电压值，前提是二极管两端电压要满足导通压降。

10.1 常见二极管类型

  • 整流二极管：整流、防反接、防倒灌。
  • 稳压二极管：反接在电路中，一般与电阻搭配使用，完成降压稳压。
  • 发光二极管：照明、指示灯。
  • 光电二极管：红外接收、遥控。
  • 瞬态抑制二极管：反接，吸收尖峰脉冲、防静电。

10.2 整流二极管

  • 作用：整流、防反接、防倒灌。
  • 把交流电变成直流电：因为只有一半能导通，所以另一半电流不能通过，就相当于是一个直流电。

整流方式

  • 全波整流：利用整流桥，和半波整流原理一致。
  • 半波整流：本质上只用到一半电流。配合一个电容使用，可以减少甚至消除电流断档。

10.3 稳压二极管

  • 反向击穿电压低一些，更容易利用其反向击穿后的特性。
  • 反向击穿后，不管电流是多少，只要能够承受，两端的电压都会稳定在一个固定值附近。
  • 稳压二极管如果不反接，相当于一个普通二极管。

10.4 发光二极管（LED）

  • LED 是一种将电能转化为光能的半导体器件，核心原理是电致发光效应。
  • 电致发光效应。
  • 光生伏特效应。

为什么要给发光二极管加限流电阻

  • LED 在点亮后，即使电压轻微增加，电流也会急剧增大，导致 LED 烧毁。
  • 限流电阻大小计算：

数码管

  • 共阴极、共阳极。
  • 一个时刻只能显示一位数字，通过扫描法显示多位数字。

LED 点阵屏

  • 模块化。

11. 三极管 BJT

  • 三极管由于是消耗电流型的电子开关，寄生电容带来的影响可以忽略不计。
  • 三极管要配合下拉或上拉电阻使用，N 下 P 上，防止工频干扰。
  • N 型三极管 NPN：电流由基极流向发射极。
  • P 型三极管 PNP：电流由发射极流向基极。
  • 三极管是由三个半导体层构成的电流控制型器件，可通过小电流的变化来控制大电流。

12. MOS 管

MOS 管是电压控制型电子开关。

  • 漏极：D。
  • 栅极：G。
  • 源极：S。
  • 核心：利用电压控制电流。

12.1 N 沟道与 P 沟道

N 沟道

  • G 高、S 低。
  • 电压差大于导通电压时，可以使 D-S 导通。
  • 增强型：默认 D-S 为关闭状态，电路符号为虚线。
  • 耗尽型：默认 D-S 为打开状态，电路符号为实线。

P 沟道

  • G 低、S 高。
  • 增强型。
  • 耗尽型。

12.2 类型判断

  1. 引脚：中间是 G，线最多的是 S。
  2. 沟道：箭头指向 G 是 N 沟道，箭头背离 G 是 P 沟道。
  3. 类型：虚线增强，实线耗尽。
  4. 导通条件：N 沟道 VGS > 0，P 沟道 VGS < 0。

12.3 寄生电容和寄生体二极管

寄生电容

  • G、D、S 两两之间存在寄生电容，因为三个极之间是绝缘的。
  • 产生问题：寄生电容导致电源开关无法正常控制电路的工作和断开。
  • 开关断开后，电容中的电无法通过回路进行释放，一直保持着原来电源的电压大小。它会慢慢漏掉，但持续时间较长。
  • 结果：导致 D-S 一直导通。
  • 解决：在 D-S 间添加一个电阻，作为电容的释放电路。

寄生体二极管

  • 存在于 D-S 之间。
  • NMOS 二极管方向：S -> D。
  • PMOS 二极管方向：D -> S。

12.4 振铃效应与米勒平台

  • 由于这两种影响的存在，一般给 MOS 管的栅极接一个 10 ~ 100 Ω 的电阻。

12.5 上拉电阻与下拉电阻

  • 下拉电阻：把电路中的一个点，通过电阻，接到了比这个点电位更低的地方。
  • 上拉电阻：把电路中的一个点，通过电阻，接到了比这个点电位更高的地方。
  • 上拉电阻直接与 VCC 连接。
  • 下拉电阻直接与 GND 连接。

12.6 三极管 VS MOS 管

控制方式

  • 三极管：电流控制型，用基极电流控制集电极电流，且需要持续提供基极电流。
  • MOS 管：电压控制型，用栅极和源极的电压控制漏极电流，只需要维持栅极源极电压就能导通漏极。

功耗

  • 三极管：集电极和发射极存在导通压降。
  • MOS 管：漏极与源极存在电阻，且电阻很小。
  • MOS 管功耗小于三极管。

其他

  • 三极管：开关速度略慢，热稳定性略差。
  • MOS 管中存在寄生电容带来的影响。
    • 加电阻：构建电容放电回路。
  • 三极管中存在工频干扰。
    • 加电阻：消除工频干扰。

12.7 推挽输出与开漏输出

  • 推挽输出可以向外输出电流，也可以向内拉取电流，但是不能随意更改输出电压。它可以实现上管导通输出高电平，下管导通输出低电平。
  • 开漏输出：上管恒断开，只能输出低电平。本来只有高阻态和低电平状态；高阻态时上下都断开，这时电流为 0，根据欧姆定律，可知电阻相当于无穷大。结合上拉电阻后可以输出高电平，且可以通过改变上拉电阻连接的 VCC 改变输出电压。
输出类型 写 1 写 0
推挽输出 输出高电平 输出低电平
开漏输出 输出高阻态 输出低电平

12.8 通用输出与复用输出

  • 通用输出：直接改变输出电平，输出高电平或低电平。
  • 复用输出：有其他模块接管。
